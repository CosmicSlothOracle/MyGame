# UI Music

Musik für Menü-Screens und UI-Transitions.

## Dateien

### `title_intro.mp3`

- **Verwendung:** Intro-Sound beim Title Screen
- **Typ:** One-Shot (spielt einmal durch)
- **Startet:** Bei Interaktion auf dem Title Screen
- **Endet:** Automatisch, dann Wechsel zu Character Select
- **Empfohlene Länge:** 3-8 Sekunden

### `character_select_loop.mp3`

- **Verwendung:** Background-Musik für Character Selection
- **Typ:** Loop (nahtlos wiederholbar)
- **Startet:** Nach title_intro.mp3
- **Endet:** Beim Wechsel zu Stage Select
- **Empfohlene Länge:** 30-120 Sekunden (Loop-Point beachten!)

---

## Namenskonvention

- Lowercase
- Beschreibender Name + Funktion (z.B. `_loop`, `_intro`)
